# 🛏 숙박 시설 리뷰 분석 및 자동 응답 생성 AI

LangChain의 LCEL(LangChain Expression Language)을 활용하여 숙박 시설 리뷰를 다단계로 분석하고, 고객에게 보낼 공손한 응답을 자동으로 생성하는 AI 서비스입니다.

<br/>

## 프로젝트 소개

- 외국어로 작성된 숙박 시설 리뷰를 입력받아, 번역 → 요약 → 감성 점수 평가 → 언어 감지 → 응답 생성 → 응답 번역까지 이어지는 파이프라인을 하나의 체인으로 구성했습니다.  
- 각 단계는 독립적인 프롬프트와 체인으로 분리되어 있으며, `RunnablePassthrough.assign()`을 통해 이전 단계의 출력을 다음 단계의 입력으로 자연스럽게 전달합니다.

<br/>

## 주요 기능

- **리뷰 번역**: 원본 리뷰(어떤 언어든)를 한글로 번역
- **리뷰 요약**: 번역된 리뷰 내용을 핵심만 간추려 요약
- **감성 점수 평가**: 번역된 리뷰를 바탕으로 감성(긍정/부정) 점수 산출
- **원본 언어 감지**: 리뷰가 작성된 원본 언어를 식별
- **공손한 응답 생성**: 요약된 내용을 바탕으로 숙박 시설 측의 정중한 답변 생성
- **응답 다국어 번역**: 생성된 응답을 감지된 원본 언어로 번역하여 고객이 이해할 수 있는 언어로 최종 제공
- 6개의 체인을 하나의 파이프라인으로 결합하여 입력 리뷰 하나로 전체 분석·응답 과정을 한 번에 실행


### 사용 기술
- LangChain (LCEL 기반 체인 구성)
- PromptTemplate + RunnablePassthrough.assign()
- LLM 기반 번역 / 요약 / 감성 분석 / 언어 감지 / 응답 생성

### 처리 흐름 (파이프라인)

| 단계 | 체인 | 설명 |
|------|------|------|
| 1 | 리뷰 번역 체인 | 영어로 작성된 리뷰를 한글로 번역 |
| 2 | 리뷰 요약 체인 | 번역된 리뷰를 요약 |
| 3 | 감성 점수 체인 | 번역된 리뷰의 0점에서 10점 사이의 점수 산출 |
| 4 | 언어 감지 체인 | 원본 리뷰가 작성된 언어 식별 |
| 5 | 응답 생성 체인 | 요약을 바탕으로 공손한 답변 생성 |
| 6 | 응답 번역 체인 | 생성된 답변을 한국어로 번역 |

각 체인은 `PromptTemplate`으로 정의된 프롬프트와 LLM을 파이프(`|`)로 연결한 형태이며, 전체 파이프라인은 `RunnablePassthrough.assign()`으로 이전 단계의 결과를 딕셔너리 형태로 누적하며 다음 체인에 전달합니다. 최종 실행부는 `try/except` 구문으로 감싸 오류 발생 시에도 안전하게 처리되도록 구성했습니다.
